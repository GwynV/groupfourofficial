MOBILE APPLICATION WIDGETS

    ActionBar is a toolbar that displays the activity title, navigation interfaces, and other interactive items.

    AutoCompleteTextViewis an editable text view element that shows completion suggestions automatically while the user is typing.

    The Calendar class is used for converting a specific instance in time (a millisecond value that is offset from the epoch) to values such as year, month, hour, day of the month, and the date of the next week.

    CardView is a UI component that presents text and image content in views that resemble cards.

    EditText is a UI element that is used for entering and modifying text.

    The gallery is a layout widget that is used to display items in a horizontally scrolling list; it positions the current selection at the center of the view.

    The Navigation Bar provides navigation controls on devices that do not include hardware buttons for Home, Back, and Menu.

    Pickers are UI elements that allow the user to pick a date or a time by using dialogs that are provided by Android.

    Date Picker is used to select a date (year, month, and day)

    Time Pickeris used to select a time (hour, minute, and AM/PM)
